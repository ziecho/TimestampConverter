# TimestampConverter
A simple Mac OS application is used to convert timestamp to date string

#Installation and Usage
You can clone the project, and compile and run it by Xcode.   
The application will monitor the clipboard, and when the clipboard's content is recognized as a timestamp, it will automatically be converted to date stringï¼Œand the conversion results will be displayed in the floating window.

Note: the floating window will keep staying on top of other windows.
